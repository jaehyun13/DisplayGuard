# DisplayGuard v1.1.0

**Now speaks your language.**

This release adds full multi-language support to the DisplayGuard UI.

## What's new

- 🌐 **9 languages**, auto-detected from your Windows display language:
  English · 한국어 · 日本語 · 简体中文 · Deutsch · Español · Français · Русский · Português (BR)
- 🔀 **Switch language on the fly** from the tray icon's new **Language** menu —
  no restart needed. Your choice is remembered.

Everything from v1.0.0 (automatic multi-monitor layout restore, tray, single
instance, lightweight Windows-API core) is unchanged.

## Install

1. Download **`DisplayGuard-Setup-1.1.0.exe`** below.
2. Run it — per-user install, **no admin prompt**. Optionally start with Windows.
3. Arrange your monitors, click **Save current layout**, and you're set.

> **SmartScreen:** as a new unsigned binary, Windows SmartScreen may warn on first
> run. Click **More info → Run anyway**. (Free code signing is in progress.)

## Requirements

- Windows 10 (1903+) or Windows 11, 64-bit

## Upgrading from 1.0.0

Just run the new installer — it upgrades in place and keeps your saved layout.
